  • URL

  • File

Special tags

Tags starting with : are reserved. You're discouraged of using them, but this won't be forbidden in software. Elements with such tags are usually to be made publicly available to other services.

If you need other text formatter than mdown/markdown, please let us known.

Name Description
:fortune The element is a fortune, which will be available at fortune.awesom.eu
:blog The element is a blog post, which will be available at blogs.awesom.eu/username
:tip The element is a tip, which will be available at tips.awesom.eu
:bookmark The element is a bookmarks, which will be available at bookmarks.awesom.eu
:public The element is public. It will be readable by anyone. They will thus appears in user search results.
:mdown The element should be formatted with mdown.
:markdown The element should be formatted with markdown.
